# Getting Started

Welcome to the BlackRoad documentation! This guide will help you get up and running quickly.

## Installation

To begin, install dependencies using pnpm:

```bash
pnpm install
```

## Running the Development Server

Start the development server at `http://localhost:3000`:

```bash
pnpm dev
```

Open your browser and navigate to the URL to view the site.
